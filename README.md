# 📊 Text Analyzer
Um analisador de texto simples desenvolvido em C# (.NET 9) que recebe o texto digitado pelo usuário e apresenta informações a respeito dele.

Projeto criado com o objetivo de treinar instruções e lógica em C#.

<hr>
<h2>⚙️ Funcionalidades</h2>

O programa analisa um texto digitado pelo usuário e retorna:

✔️ Quantidade total de palavras <br>
✔️ Lista de todas as palavras <br>
✔️ Quantidade total de caracteres <br>
✔️ Palavras que começam com letra maiúscula <br>
✔️ Palavras que começam com letra minúscula <br>
✔️ Palavras que começam com números <br>

<h2>✏️ Conceitos aplicados </h2>

✔️ Manipulação de string <br>
✔️ Split() para separação de palavras <br>
✔️ Coleção de objetos List<T> <br>
✔️ Estruturas de repetição (foreach) <br>
✔️ Condicionais (if) <br>
✔️ Métodos  <br>
✔️ Encapsulamento em classes <br>
✔️ Manipulação de caracteres (char) <br>

<h2>🛠 Tecnologias</h2>

• C# <br>
• .NET 9 <br>
• Console Application <br>
• Visual Studio 2026 <br>


<h2>▶️ Como executar o projeto</h2>
📍 Clonar o repositório <br>
git clone https://github.com/seu-usuario/textanalyzer.git <br><br>
📍 Entrar na pasta com um terminal (cmd / bash) <br>
cd textanalyzer <br><br>

dotnet run
